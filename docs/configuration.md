# Configuration

Main config file: `config/documentator.php`.

## OpenAPI Metadata

```php
'openapi_version' => '3.0.3',

'info' => [
    'title' => env('OPENAPI_TITLE', 'API Documentation'),
    'version' => env('OPENAPI_VERSION', '1.0.0'),
    'description' => env('OPENAPI_DESCRIPTION'),
    'contact' => [
        'name' => env('OPENAPI_CONTACT_NAME'),
        'email' => env('OPENAPI_CONTACT_EMAIL'),
        'url' => env('OPENAPI_CONTACT_URL'),
    ],
],
```

## Output

```php
'output' => [
    'path' => env('OPENAPI_OUTPUT', 'docs/openapi.json'),
    'format' => env('OPENAPI_OUTPUT_FORMAT', 'json'),
    'pretty_print' => env('OPENAPI_PRETTY_PRINT', true),
],
```

> [!NOTE]
> The generator command currently writes JSON output.

> [!WARNING]
> `output.format` exists in config, but `docs:generate` serializes output via `json_encode(...)` and writes JSON.

## Servers

```php
'servers' => [
    [
        'url' => env('APP_URL', 'http://localhost'),
        'description' => 'Development server',
    ],
],
```

## Security Schemes

```php
'security' => [
    'schemes' => [
        'bearerAuth' => [
            'type' => 'http',
            'scheme' => 'bearer',
            'bearerFormat' => 'JWT',
        ],
    ],

    'default' => ['bearerAuth'],
],
```

Use `@unauthenticated` in PHPDoc for public endpoints.

## Route Discovery

```php
'routes' => [
    'include' => ['api/*'],
    'exclude' => ['sanctum/*', 'horizon/*', 'telescope/*'],
    'exclude_middleware' => [],
    'methods' => ['GET', 'POST', 'PUT', 'PATCH', 'DELETE'],
],
```

> [!WARNING]
> Routes handled by closures are skipped by documentation parsing. Use controller actions for endpoints you want documented.

## Formats

```php
'format' => env('OPENAPI_FORMAT', 'simple'),

'custom_formats' => [
    // 'telegram' => App\OpenApi\Formats\TelegramFormat::class,
],
```

See [Response Formats](response-formats.md).

## Resources

```php
'resources' => [
    'namespaces' => [
        'App\\Http\\Resources',
        'App\\Models',
    ],
    'suffix' => 'Resource',
],
```

## Examples

```php
'examples' => [
    'enabled' => true,
    'locale' => env('FAKER_LOCALE', 'en_US'),
    'seed' => null,
    'collection_size' => 2,
],
```

> [!NOTE]
> Faker-based examples are generated by `SchemaBuilder` for response payload previews.

## Tags

```php
'tags' => [
    'auto_generate' => true,
    'definitions' => [
        // 'Users' => 'User management and profile operations',
    ],
    'sort' => true,
],
```

## Default Responses

```php
'responses' => [
    'default' => [
        // '401' => ['$ref' => '#/components/responses/Unauthorized'],
    ],
],
```

## Localization

```php
'locale' => env('OPENAPI_LOCALE', 'en'),
```

## Advanced

```php
'advanced' => [
    'cache_parsed_files' => true,
    'include_deprecated' => false,
    'validate_schemas' => env('OPENAPI_VALIDATE', true),
    'deep_scan_controllers' => true,
```
